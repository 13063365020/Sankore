Compress a source directory in a zip file.

  • pDir the directory to add in zip
  • pDestPath the path inside the zip. Attention, if path is not empty it must end by a /.
  • pOutZipFile the zip file we want to populate with the directory
  • UBProcessingProgressListener an object listening to the compression progress
    Returns
    bool. true if compression is successful.
